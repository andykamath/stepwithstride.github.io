This is STRIDE

STRIDE is a mobility aid wearable for the visually impaired that aims to help
them navigate areas with many obstacles at foot level. STRIDE aims to detect
objects at foot level and alert the user of their existance.

How does it work?

Accelerometer

When the user is walking with STRIDE on, the accelerometer built in detects if a step is being taken, and starts searching.

Proximity sensor

After detecting that step has been taken, STRIDE uses a proximity sensor to locate the nearest object.

Digispark

The information from the proximity sensor is relayed to the Digispark where analysis of the data is done.

Vibrator

Once the nearest object is analyzed, STRIDE relays the information to the user via a vibrator on a ankle strap.
